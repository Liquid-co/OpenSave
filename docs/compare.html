The honest version

What each tool is actually for.

Steam Cloud

Excellent when it works — zero setup, built into Steam. The catch: it only covers games bought on Steam, and only the ones whose developer opted in. Everything else (emulators, GOG, Epic, itch, that one indie gem) gets nothing.

Ludusavi

A genuinely excellent backup/restore tool with the best save-location database around — OpenSave's auto-scan is built directly on Ludusavi's community-maintained manifest, so credit where it's due. What it doesn't do is live sync: you run a backup, then manually move or restore it. If you just want reliable local backups across a huge library, Ludusavi is great at exactly that.

GameSave Manager

A mature Windows tool for organizing and versioning save backups across a big game library, with its own broad detection database. Like Ludusavi, it's a backup/restore manager, not a live sync daemon — there's no automatic peer-to-peer propagation the moment a save changes.

OpenSave

The only one of the four built specifically for live, automatic, peer-to-peer sync between your own devices — watch → snapshot → sync, the instant a save changes, with no cloud account required. Think less "backup tool," more "Steam Cloud, but for every game you own and every device you own."
